# Velora Handmade

A premium, luxury-minimalist landing page and shop for handmade crochet creations. Built with React, Vite, Tailwind CSS v4, and Framer Motion.

## Features
- **Minimalist Classic Design**: High-end fashion editorial aesthetic.
- **Dark/Light Mode**: Fully responsive seamless theme switching.
- **Framer Motion Animations**: Sophisticated cubic-bezier entrance animations and advanced loading screen.
- **WhatsApp Integration**: Direct ordering through WhatsApp with pre-filled product details and image previews.

## Tech Stack
- React
- Vite
- Tailwind CSS v4
- Framer Motion
- Zustand
- Lucide React

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
